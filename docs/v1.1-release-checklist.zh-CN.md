# v1.1 上线检查清单

更新日期：2026-03-31

本文档用于约束 `foggy-odoo-bridge` 社区版 `v1.1` 的上线前检查范围。  
它不是发布承诺，而是一个可持续补充的检查骨架。

当前判断：

- `v1.1` 功能开发已接近完成
- 但测试、验证、体验调优还没有系统完成
- 在这些工作完成前，不建议把 `v1.1` 视为正式可发布状态

## 1. 版本目标

`v1.1` 目标应保持克制：

- 社区版可跑通
- 模型权限与行权限链路成立
- MCP 主流程可验证
- 明确确认社区版不内置 AI Chat
- 文档、安装、演示路径清晰

不应在 `v1.1` 承诺：

- 列权限 / 字段脱敏
- 复杂导出
- 企业级审计与治理增强

## 2. 当前已知缺口

以下缺口当前默认视为未完成，需在发布前逐项确认：

- 自动化测试还未完整执行
- 端到端验证还未系统记录
- 管理员配置路径还未做完整体验走查
- 错误提示与边界条件还需人工复查
