# Community Docs

This directory keeps public-facing technical notes for the Community Edition.

Current documents:

- `permission-field-mapping-reference.md`: explains how Odoo `ir.rule` fields are mapped into Foggy DSL fields and why that matters for fail-closed permission enforcement.
- `v1.1-release-checklist.zh-CN.md`: release readiness checklist skeleton for the Community Edition `v1.1`.

Internal planning notes, release working files, and cross-team coordination reports are intentionally kept out of the Community Edition repository.
